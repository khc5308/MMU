# Fix The MMU

모든 음표를 수집하고 고장 난 MMU를 수리하여 가상의 공간 **MIXXTOPIA**로 향하는 Pygame 기반의 2D 탑다운 우주 모험 게임입니다.

---

## 🎮 게임 규칙 및 조작 방법

- **목표**: 2000x1500 크기의 넓은 우주 맵을 탐험하며 흩어져 있는 6개의 컬러 음표(Note)를 모두 수집하고, 맵 중앙에 위치한 MMU로 돌아와 수리 작업을 완수(엔진 가동)하십시오.
- **조작키**:
  - **이동**: `W`, `A`, `S`, `D` (상, 하, 좌, 우)
  - **치트키**: `C` (누르면 즉시 모든 음표가 수집된 상태로 처리됩니다)
  - **종료**: 성공 화면에서 `ESC` 키를 누르면 게임을 종료합니다.
- **성공 조건**:
  1. 화면 왼쪽 상단의 UI 슬롯에 6개 음표의 불빛이 모두 들어오게 합니다.
  2. 음표를 모두 수집한 뒤 맵 중앙 근처(X: 1000, Y: 750 부근)의 MMU 장치로 이동합니다.
  3. MMU에 도달하면 엔진 시동(1.5초간 화면 흔들림 연출)이 발생한 후 최종 성공 화면으로 전환됩니다.

---

## 🚀 실행 방법

### 1. 가상환경 및 의존성 설치
본 프로젝트는 외부 라이브러리로 `pygame`을 사용합니다. 가상환경을 구성하고 설치하는 순서는 다음과 같습니다.

**Windows (PowerShell/CMD):**
```bash

python -m venv venv

# 가상환경 활성화
.\venv\Scripts\activate

# 패키지 설치
pip install -r requirements.txt
```

**macOS / Linux:**
```bash
# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화
source venv/bin/activate

# 패키지 설치
pip install -r requirements.txt
```

### 2. 게임 실행
가상환경이 활성화된 상태에서 아래 명령어로 게임을 실행합니다.
```bash
python main.py
```
